# E-commerce Customer Analytics Portfolio

This portfolio presents a series of e-commerce customer analytics projects built with a 7M-row customer behavior dataset. The analysis focuses on understanding customer behavior, product performance, retention patterns, conversion funnels, and conversion prediction.

## Project Overview

This portfolio includes four connected analytics projects:

1. Customer segmentation analysis
2. Cohort and retention analysis
3. Conversion funnel analysis
4. Conversion prediction modeling

Together, these projects demonstrate my ability to clean and analyze large-scale behavioral data, translate data into business insights, and support decision-making through customer analytics.

## Projects

### 1. E-commerce Customer Segmentation Analysis

This project analyzes customer behavior and groups users based on purchasing and engagement patterns. The goal is to identify meaningful customer segments and understand how different groups contribute to business performance.

**Key Focus:**
- Customer behavior analysis
- RFM-style segmentation
- Product and customer value comparison
- Business insight generation

### 2. E-commerce Cohort & Retention Analysis

This project examines how customer groups behave over time. The analysis focuses on customer retention, repeat activity, and long-term engagement patterns.

**Key Focus:**

- Customer lifecycle insights

### 3. E-commerce Conversion Funnel Analysis

This project analyzes the customer journey from product views to purchases. The goal is to identify where users drop off and where conversion opportunities exist.

**Key Focus:**
- Funnel analysis
- View-to-cart-to-purchase behavior
- Drop-off identification
- Conversion improvement insights

**Key Focus:**
